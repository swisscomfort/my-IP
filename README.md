# Governance-IP Dashboard

Private Arbeitsfläche zur Entscheidung über Standard-Opportunitäten.

## Installation

```bash
npm install
```

## Starten

```bash
npm run dev
```

Öffne http://localhost:5173

## Build

```bash
npm run build
```

## Referenzwerk

https://github.com/swisscomfort/governance-ip

## Funktioniert offline

Alle Entscheidungen werden lokal im Browser gespeichert (localStorage).
